# StepUP - Premium Sneaker & Footwear Store

Step into your style with **StepUP**, a modern and minimalist sneaker store built with Next.js 16 and Tailwind CSS v4.

## Features
- **Clean Architecture**: Modular component-based structure.
- **Premium Design**: High-contrast, minimalist black and white aesthetic.
- **Responsive**: Fully optimized for mobile, tablet, and desktop.
- **Modern Tech**: Built using the latest Next.js App Router and Tailwind v4.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack
- [Next.js](https://nextjs.org)
- [Tailwind CSS](https://tailwindcss.com)
- [Lucide React](https://lucide.dev)
